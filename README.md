# libopencm3-examples
Collection of snnipets with libopencm3 to run in a stm32f103c8t6 (BluePill). Every example is contained in a PlatformIO project.

### Examples

 - [Blink](blink)
 - [Button](button)
 - [SysTick](systick)
 - [Tasks](tasks)
 - [Interrupt](interrupt)
 - [Timer](timer)
 - [PWM](pwm)
 - [Encoder](encoder)
 - [TB6612FNG](tb6612fng)
